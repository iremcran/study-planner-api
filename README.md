# Study Planner API

This is a simple backend project built using Java Spring Boot and MySQL.

## Features
- Add study data
- List all studies
- REST API structure
- Database connection with MySQL

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Endpoints

### GET /hello
Returns a test message

### GET /all
Returns all study records

### POST /add
Adds a new study

Example JSON:
{
  "name": "Math",
  "hours": 3
}
